## 프로잭트 개요 

--
사람들에게 일정 관리는 귀찮을 경우가 많습니다. 직접 입력하는 방식을 사용하다보니 늦게 입력할 경우 내용을 잊어버리는 경우가 많고, 제대로된 일정을 기록하지 못하는 경우가 종종 생기기도 합니다. 이로 인해 일정에 차질이 생기거나 불편한 일이 생길 수도 있습니다. 또한 회의 중에 불규칙적으로 발생하는 차후 회의 계획 등의 일정은 일일히 따로 기록한 후 추가하는 것이 번거로우면서도 예측 불가능한 이유로 미기재 시 치명적인 결과를 불러 일으킬 수 있습니다.

이로인해 SpeakPlan은 음성을 인식하여 자동으로 예고된 일정을 추출하여 캘린더에 자동으로 기재되는 AI 캘린더입니다. 현장 대화나 회의록 등 많은 대화나 텍스트에서 일정 및 장소를 추출하여 캘린더에 자동 기재해줍니다. 음성 파일(녹음)의 경우에는 STT를 통해 인식을 하고, STT를 통해 인식한 데이터와 회의록 등 텍스트 파일을 Gemini API를 사용하여 내용 중 장소, 일정, 시간 등 특정 키워드를 추출하고 구글 캘린더와 연동돠어 구글 캘린더에 일정을 기재합니다.

## 이용 방법
-
1. 로그인 서비스 이용
   1) 구글 OAuth와 연동하여 구글 아이디로 로그인합니다.
   2) 본인의 구글 캘린더에 저장되어 있는 기록들을 불러옵니다.

2. 대화 녹음
   1) 대화를 시작할 때 녹음 버튼인 'ON AIR'를 눌러주세요
   2) 대화 내용이 자동적으로 녹음됩니다.
   3) 대화가 끝났을 때 다시 한번 녹음 버튼인 'ON AIR'를 눌러주세요ㄴ
   4) 녹음이 종료되면 자동적으로 녹음 파일이 저장됩니다.

3. 텍스트 파일 및 음성 파일 전송
   1) 핸드폰에 저장된 파일 중 추출이 필요한 파일을 선택합니다.
   2) 선택된 파일을 전송합니다.
   3) 전송된 파일은 STT와 Gemini를 사용하여 일정 및 장소 등 특정 키워드를 추출

4. 캘린더에 업로드
   1) STT와 Gemini를 사용하여 추출된 일정 키워드를 받아옵니다.
   2) 받아온 키워드들을 팝업창에서 확인을 합니다.
   3) 수정이 필요한 부분을 체크하여 직접 수정합니다
   4) 완료 버튼을 누르게 되면 자동적으로 캘린더에 정보가 업로드 됩니다.
-

1. 음성을 텍스트로 변환
2. 텍스트를 Gemini로 데이터 추출( 시간(년/월/일/시), 요약, 장소),
3. 데이터를 Google Calendar에 저장
4. 저장한 데이터 알림
- (선택사항) 중요도에 따른 알림 중요도 표시, 시전 조절
